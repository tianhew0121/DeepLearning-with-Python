## Getting Started


### Prerequisites
Install numpy if you haven't.
* numpy
  ```sh
  pip install numpy
  ```

Install h5py if you want to run the demo.
* h5py
  ```sh
  pip install h5py
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/tianhew0121/DeepLearning-with-Python.git
   ```
2. Simply run with Python.
   ```sh
   python3 neural_network.py
   ```
3. Done!
